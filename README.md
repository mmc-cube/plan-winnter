# 寒假任务管理器 🎯

一个功能丰富的本地部署寒假任务管理网页应用，帮助用户高效安排时间，通过可视化时间表、任务拖放、时间追踪等功能提升学习和生活效率。

## ✨ 主要功能

### 🎨 卡通风格界面
- 活泼有趣的卡通设计风格
- 丰富的emoji表情和动画效果
- 响应式设计，适配桌面和移动设备

### 📅 可视化时间表
- 24小时时间段显示
- 拖放式任务分配

### 📋 灵活的任务管理

### ⏱️ 时间追踪与计时
- 精确的任务计时功能
- 累计时间统计
- 实时计时器显示
- 暂停/继续/停止计时控制

### ⏰ 假期倒计时
- 显眼的假期剩余天数提醒
- 实时更新倒计时
- 动画效果增强视觉吸引力

## 📖 使用指南

### 基本操作

1. **设置假期结束日期**
   - 编辑 `js/app.js` 文件中的 `vacationEndDate` 变量
   - 格式：`new Date('YYYY-MM-DD')`

2. **添加任务到时间表**
   - **方法一**：从左侧任务库拖拽任务到对应时间段
   - **方法二**：点击时间段，在弹窗中查看和管理任务

3. **自定义任务**
   - 在任务库底部输入框中输入任务名称
   - 点击"添加"按钮或按Enter键
   - 新任务会出现在任务库中，可以拖拽使用

4. **时间追踪**
   - 点击已分配任务的时间段
   - 在弹出的任务详情窗口中点击"开始计时"
   - 使用暂停/继续/停止按钮控制计时
   - 累计时间会自动保存和显示

5. **日期导航**
   - 使用"前一天"/"后一天"按钮切换日期
   - 每天的任务安排独立保存

### 高级功能

1. **数据管理**
   ```javascript
   // 在浏览器控制台中使用以下命令

   // 导出数据
   WinterVacationUtils.exportData();

   // 清除所有数据
   WinterVacationUtils.clearAllData();
   ```

2. **自定义假期结束日期**
   ```javascript
   // 在 js/app.js 中修改
   this.vacationEndDate = new Date('2025-02-17'); // 改为你的假期结束日期
   ```

## 🔧 自定义配置

### 修改预设任务
在 `js/app.js` 中的 `getTaskEmoji` 方法中添加或修改：

```javascript
getTaskEmoji(taskName) {
    const emojiMap = {
        '学习': '📖',
        '运动': '🏃‍♂️',
        '阅读': '📚',
        '编程': '💻',
        '休息': '😴',
        '娱乐': '🎮',
        // 添加你的自定义任务
        '绘画': '🎨',
        '音乐': '🎵'
    };
    return emojiMap[taskName] || '📝';
}
```

### 修改主题颜色
在 `css/style.css` 中的 `:root` 选择器中修改CSS变量：

```css
:root {
    --primary-color: #FF6B9D;    /* 主色调 */
    --secondary-color: #4ECDC4;  /* 辅助色 */
    --accent-color: #FFE66D;     /* 强调色 */
    /* 修改为你喜欢的颜色 */
}
```
